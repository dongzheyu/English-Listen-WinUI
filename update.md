# English Listen 更新日志

## 版本 1.1.0

### 新功能
- **背单词界面**: 新增独立的背单词页面，支持单词浏览和中英文朗读
- **重读功能**: 背单词页面添加重读按钮，可反复听取单词发音
- **在线听写改进**: 纸笔模式现在正确保存测试结果到历史记录

### 功能改进
- 翻译限额调整为每日 1000 个单词
- 离开单词页面返回后自动保存功能恢复正常
- 二次进入听写页面不再丢失语音合成器
- 朗读中断功能优化，停止/跳过操作立即生效
- 优化异步异常处理，避免静默丢失的异常

### 问题修复
- 修复了 WordsPage 缓存返回后 Timer 被销毁不重建的问题
- 修复了 DictationTestPage 纸笔模式结果永不保存的问题
- 修复了 DictationTestPage 二次进入后 synthesizer 为 null 的问题
- 修复了 speakTaskSource 在异步期间被 Dispose 设为 null 导致的潜在空引用
- 修复了 SpeechService.SpeakAsync 的 CancellationToken 被完全忽略的问题
- 修复了 Submit 方法为 async void 导致异常静默丢失的问题
- 修复了纸笔模式 NumberBox 绕过验证后的递归栈溢出问题
- 修复了 ModernDictationService.Dispose 未解绑 Elapsed 事件导致的内存泄漏
- 修复了首次启动时 ProgressPage.TestHistory 为 null 导致的崩溃
- 修复了硬编码 UI 控件索引的问题
- 修复了使用字符串类型名检查参数的问题
- 修复了 WordsPage 过滤后编辑单词，重置过滤显示旧数据的问题
- 修复了首次安装无 API 密钥时进入设置页崩溃的问题
- 修复了 SecretStorageService 缺少 System.Security.Cryptography.ProtectedData 包引用
- 修复了 TempFileHelper 和 BaiduTranslateService 缺失 System.Linq 引用
- 移除了所有界面中的 emoji 字符

## 版本 2.7.0 (WinUI3版本)

### 新功能
- **全新界面架构**: 基于 WinUI3 框架重构，采用现代化的 Windows 原生界面设计
- **NavigationView 侧边栏**: 实现侧边栏导航，包含主页、查看单词、用户管理、学习进度、帮助、关于等页面
- **Windows 11 语音支持**: 支持 Windows 11 24H2+ 系统自带的高质量语音合成
- **智能语音引擎**: 自动检测系统版本并选择最佳语音引擎（Windows SAPI 或 Windows TTS）
- **语音试听功能**: 新增 Windows TTS 语音的试听功能，可预览不同语音效果
- **应用内更新系统**: 完整的更新检查、版本比较、下载和安装功能
- **MVVM 架构**: 采用 Model-View-ViewModel 架构设计，提升代码可维护性
- **混合语言开发**: C# 前端 + C++ 原生后端，兼顾开发效率和性能

### 功能改进
- **性能优化**: 使用 WinRT API 提升语音合成性能
- **兼容性增强**: 同时支持 Windows 10（版本 17763）和 Windows 11 系统
- **配置管理**: 所有设置自动保存到 JSON 配置文件
- **错误处理**: 增强的异常处理机制和用户反馈
- **主题系统**: 支持深色/浅色主题切换，自动适配系统主题
- **用户账户**: 完整的多用户管理和密码保护系统
- **词库管理**: 支持词库分组、搜索筛选、导入导出等功能
- **学习统计**: 可视化图表展示学习进度和趋势，支持导出统计数据
- **快捷键支持**: 测试界面支持空格键重复、方向键导航、ESC 键暂停/继续

### 技术架构
- **编程语言**: C# 12 + C++17
- **GUI 框架**: WinUI3
- **运行时**: .NET 8.0
- **Windows App SDK**: 1.8+
- **构建系统**: MSBuild + CMake（用于 Native Backend）

### 页面结构
- **HomePage**: 主页，提供快速开始和使用技巧
- **WordsPage**: 单词管理页面，支持词库编辑、导入导出、分组管理
- **DictationTestPage**: 听写测试页面，支持在线模式和纸笔模式
- **AnswersPage**: 答案显示页面
- **ProgressPage**: 学习进度页面，展示统计数据和历史记录
- **SettingsPage**: 设置页面，包含主题、语音、更新等配置
- **UserPage**: 用户管理页面，支持登录、创建用户
- **HelpPage**: 帮助页面，提供使用教程
- **AboutPage**: 关于页面，显示版本信息和许可证

### 问题修复
- 修复了设置界面崩溃问题
- 优化了语音引擎切换逻辑
- 改进了版本检测和更新机制
- 修复了词库文件保存路径问题
- 优化了内存使用和性能

### 说明
这是从 Qt6 版本移植到 WinUI3 的重要更新，带来了更好的用户体验和更现代的界面设计。移除了例句学习功能，专注于核心的单词听写训练。
