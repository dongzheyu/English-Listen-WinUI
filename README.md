# English Listen - WinUI3

English Listen 是一个基于 WinUI3 框架开发的英语听写练习工具，帮助用户通过听写训练提升英语词汇记忆和听力能力。

## 功能特性

### 词库管理
- 自定义词库：自由添加、编辑、删除单词列表
- 词库分组：支持将词库按主题或难度分组管理
- 文件导入导出：支持从文本文件导入词库，导出学习成果
- 高级筛选：按字母、数字开头等条件筛选单词

### 听写模式
- 纸笔听写模式：传统听写方式，适合专注练习
- 在线听写模式：实时输入答案并自动检查正确性
- 可调节间隔：朗读时间间隔可在1-60秒间自由调整
- 随机播放：支持随机顺序播放单词，增加挑战性

### 语音引擎
- Windows SAPI：使用系统内置语音引擎，稳定可靠
- Flite 引擎：集成第三方 Flite 语音引擎，提供更多语音选择
- 多语音模型：支持多种 Flite 语音模型（女声/男声）
- 语音试听：可预览不同语音效果，选择最适合的发音

### 用户体验
- 主题切换：支持深色/浅色主题，保护眼睛
- 流畅界面：现代化 WinUI3 界面设计
- 响应式布局：适配不同屏幕尺寸
- 快捷键支持：空格键重复、方向键导航等

### 学习统计
- 测试历史：完整记录每次测试的详细数据
- 准确率图表：可视化展示学习进步趋势
- 学习进度：统计总测试次数、平均正确率等指标
- 连击记录：追踪连续学习天数

### 数据安全
- 用户账户：支持多用户管理和密码保护
- 数据加密：敏感数据加密存储，保护隐私
- 本地存储：所有数据默认保存在本地，无需网络

### 隐私控制
- 数据收集开关：完全控制是否允许数据收集
- 云同步选项：可选择是否启用云同步功能
- 统计分析控制：决定是否参与匿名统计分析
- 学习数据分享：自主选择是否分享学习统计数据

## 系统要求

- 操作系统：Windows 10 版本 17763 或更高版本
- 运行时：.NET 6.0 或更高版本
- 框架：Windows App SDK 1.5 或更高版本

## 快速开始

### 安装步骤
1. 下载最新版本的安装包
2. 双击安装程序并按照提示完成安装
3. 启动 English Listen 开始您的英语听写练习

### 从源码编译
如果您希望从源码编译或进行开发：

1. 克隆项目到本地：
   ```bash
   git clone https://github.com/your-username/English-Listen-WinUI.git
   ```

2. 使用 Visual Studio 2026 打开解决方案文件

3. 恢复 NuGet 包依赖：
   ```bash
   dotnet restore
   ```

4. 编译并运行项目：
   ```bash
   dotnet build
   dotnet run
   ```

## 文件位置

应用的所有文件（包括配置文件和词库文件）都存储在软件安装目录中，便于管理和备份：

- settings.json - 主要设置文件
- wordlist_groups.ini - 词库分组配置
- test_history.json - 测试历史记录
- wordlist/ 目录 - 词库文件存储目录
